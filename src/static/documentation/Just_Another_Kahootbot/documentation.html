Before diving deeper, it's important to understand how the bot is structured. The bot is divided into two main components:

Main Parts

  • Main Bot: This is the component you interact with. It joins games, clicks buttons, and makes intelligent decisions about what to do next. It utilizes Selenium, as mentioned in the installation guide.
  • Answer Getter: This component fetches and parses game statistics from the Kahoot website to retrieve the answers. It uses BeautifulSoup, as you'll likely have seen in the installation guide.

Installation

To get the project up and running, follow these steps:

  1. Clone the repository.
  2. Install the required dependencies.
  3. Run the project.
